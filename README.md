UC4: Maintain Ordered Bogie Consist (LinkedList)

Description
This use case models the physical sequence of bogies using LinkedList.

What the program does
- Creates a LinkedList representing the train
- Adds bogies in order
- Inserts a bogie at a specific position
- Removes bogies from the start and end
- Displays the final consist

Concepts Used
- LinkedList
- addFirst()
- addLast()
- add(index, element)
- removeFirst()
- removeLast()

Outcome
Shows how LinkedList maintains ordered structures.
