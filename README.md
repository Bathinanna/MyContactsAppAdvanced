UC2: Add Passenger Bogies to Train (ArrayList Operations)

Description
This use case demonstrates how passenger bogies can be dynamically added and removed using ArrayList.

What the program does
- Creates a list of passenger bogies
- Adds bogies such as Sleeper, AC Chair, and First Class
- Displays the list
- Removes a bogie
- Checks if a bogie exists using contains()

Concepts Used
- ArrayList
- add()
- remove()
- contains()
- Dynamic list management

Outcome
Shows how collections can manage train bogies dynamically.
