UC1: Initialize Train and Display Consist Summary

Description
This use case initializes the Train Consist Management App and displays the initial state of the train.

What the program does
- Starts the application
- Prints a welcome message
- Creates an empty train consist
- Displays the initial bogie count

Concepts Used
- Java Class
- main() method
- static keyword
- ArrayList
- List interface
- Console output using System.out.println()

Outcome
The train consist is created but contains no bogies yet.
