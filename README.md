UC3: Track Unique Bogie IDs (HashSet)

Description
This use case ensures that duplicate bogie IDs cannot be added to the train.

What the program does
- Creates a HashSet for bogie IDs
- Adds multiple IDs including duplicates
- Displays the final set

Concepts Used
- Set interface
- HashSet
- Automatic duplicate removal
- add()

Outcome
Demonstrates how HashSet guarantees uniqueness.
